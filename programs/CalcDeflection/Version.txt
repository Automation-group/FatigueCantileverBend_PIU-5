Программа CalcDeflection предназначена для расчёта прогиба образца
при заданном механическом напряжении. Данный расчёт требуется
для задания смещения образца при испытании материаллов на усталость.

Версии программы CalcDeflection

Версия 0.2
 - Добавлена возможность расчёта при разном направления приложения сила -
   по оси X или по оси Y.
 - Добавлена возможность расчёта для поводка круглой формы.
 - Добавлены дополнительные настройки вывода результата расчёта.
 - Добален выбор шага интегрирования, как степени числа 10. По умолчанию
   шаг интегрирования равен (Ls+Lp)/1.0e+6. Величину шага можно изменять
   в диапазоне от 1.0e+2 до 1.0e+8.
 - Добавлен выбор размера и параметров шрифта для всей программы.
   В качестве базового шрифта программа берёт системный шрифт который
   выставлен по умолчанияю в операционной системе.
 - Добавлено окно с информацией о программе. (планируется)
 - Добавлена описание алгоритма расчёта в главном меню
   Справка -> Методика расчёта. (планируется)

Версия 0.1
 - Разработан алгоритм расчёта прогиба для образца с радиусной рабочей
   частью и поводком прямоугольного сечения.
 - Разработан интерсейс на Qt5/Qt6 для удобства расчёта.
